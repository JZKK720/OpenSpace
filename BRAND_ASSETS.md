## Branded Presentation Assets

The following types of materials should also be treated as brand-restricted when they are used to identify the official Cubecloud product, service, or marketing presentation:

- repository hero graphics and branded screenshots
- product badges, wordmarks, and icon treatments
- marketing images that present the official Cubecloud UI or identity as a source identifier

Examples in this repository include branded README presentation assets under [assets](assets).

## What This Means

You may keep these files in the repository for use with the official Cubecloud project and for normal contribution workflows in this repository.

You may not reuse these brand assets to market, package, or present an independent fork as Cubecloud without written permission.

If you publish a public fork, replace Cubecloud-specific logos and source-identifying brand assets unless you have written permission to retain them.
